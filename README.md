# pyflask-uploadfiles  
基于pyflask的文件上传, 下载网页, 目前仅实现基本功能, 较为简陋  

## 运行方法  
在同一端口下, 先运行网页/index.html, 后运行pyflask_uploadfile.py, 添加文件或文件夹后上传, 上传后的文件存储在floder文件夹内.   
可以下载上传的文件.  
由于兼容性问题, 过老的设备可以访问/simple.html网页进行文件上传.   

## 注意事项  
未对文件名进行安全检查.  
同文件名文件会覆盖.
